
  # Wanderlust Unified Vite

  This is a standalone merged project built on the Wanderlust Architects WebApp stack.

  It uses:
  - the Vite + React + Tailwind + React Router architecture from `Wanderlust Architects WebApp`
  - the marketing-site design direction, layout patterns, and editorial content structure from `Wanderlust Architect Website`
  - the WebApp projects experience as the standalone `/projects` flow

  ## Running the code

  Run `npm install` to install dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build.
